AEDS1

Este repositório reúne os códigos desenvolvidos durante a disciplina de Algoritmos e Estruturas de Dados I (AEDS1). Ele foi criado como um espaço pessoal para armazenar, organizar e revisar exercícios, listas, provas e demais materiais produzidos ao longo da faculdade.

Objetivo

O objetivo deste repositório é servir como um arquivo de estudo e revisão, reunindo implementações e resoluções que podem ser úteis para consultas futuras. Não se trata de um projeto finalizado, mas de um conjunto de códigos que acompanharam o processo de aprendizado ao longo da disciplina.

Conteúdo

* Este repositório contém:

* Códigos em C e C++ utilizados nas aulas;

* Listas de exercícios;

* Revisões de prova e simulados;
